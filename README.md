# 📖 Overview

Nova Calculator is a modern scientific calculator application designed to deliver a premium calculation experience with a beautiful glassmorphism interface.

The application supports basic arithmetic operations, advanced scientific functions, keyboard interactions, degree/radian calculations, and a responsive UI optimized for desktop and mobile users.

It focuses on accuracy, speed, accessibility, and clean user interaction.
